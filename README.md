# Dé 3D — React Three Fiber
Simulation d'un lancer de dé en 3D dans le navigateur

## Fonctionnalités
- Dé 3D avec textures personnalisées sur chaque face
- Rotation animée vers la face correspondant au résultat
- Bouton de lancer tirant un nombre aléatoire entre 1 et 6
- Historique des lancers persisté en localStorage


## Technologies
- React
- React Three Fiber

## Installation

```bash
npm install
npm start
